# CyberNotes

This repository is for organizing and uploading my cybersecurity notes, lab work, summaries, and study material.

## Purpose

- Keep semester notes in one place
- Track cybersecurity concepts and tools
- Store lab writeups and practice material
- Build a personal reference for exam preparation and revision

## Suggested Structure

```text
CyberNotes/
├── README.md
├── Topics/
│   ├── Networking/
│   ├── Cryptography/
│   ├── Web-Security/
│   ├── Malware-Analysis/
│   ├── Digital-Forensics/
│   └── Ethical-Hacking/
└── Resources/
    ├── CheatSheets/
    ├── Tools/
    └── References/
```

## What I Will Upload Here

- Class notes
- Lab tasks and solutions
- Important definitions and short summaries
- Command references
- Practice questions
- Useful cybersecurity resources

## Notes Format

You can upload notes in any format you prefer, for example:

- `.md` for clean GitHub-readable notes
- `.pdf` for exported lecture notes
- `.docx` for editable documents
- `.png` or `.jpg` for diagrams and handwritten notes

## Recommended Naming

- `Lecture-01-Introduction.md`
- `Lab-02-Nmap-Scanning.md`
- `Assignment-01.pdf`
- `Cryptography-Basics.md`

## Goal

The goal of this repository is to maintain a clean, searchable, and well-organized cybersecurity notes archive that can grow over time.
